# 2025 Exo Moustache 
The Moustache is a small pcb made to receive I2C in differential, CAN and power by an ethernet port and distributed trought 6 other ethernet port to connect the captors and motors.
## PIN Layout
Pin 1: CAN+

Pin 2: CAN-

Pin 3: SCL+

Pin 4: SCL-

Pin 5: SDA+

Pin 6: SDA-

Pin 7: 5V

Pin 8: GND
## Port Layout
MOTG = moteur Genou

MOTH = moteur Hanche

BNOF = BNO Femur

BNOT = BNO Tibia

EXT = extra
